# Coin-Toss-Game

A simple Python game where players guess whether a coin toss will land on **Heads** or **Tails**.

## Features

- Single-player mode
- Two-player mode
- Random coin toss simulation
- Score tracking across multiple rounds
- Input validation to prevent invalid choices

## How It Works

1. The player chooses a game mode:
   - Single Player
   - Two Players

2. Players guess:
   - `h` for Heads
   - `t` for Tails

3. The program generates a random coin toss result.

4. The program compares player guesses with the generated result.

5. Scores are updated and displayed after each round.

6. Players can choose to continue or exit the game.

## Technologies Used

- Python 3
- `random` module
